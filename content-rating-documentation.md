# Content Rating Documentation for Heavenly Guidance

## App Overview
Heavenly Guidance is an educational and entertainment app that provides users with:
- Constellation identification and mythology
- Astrological information
- Personalized horoscope readings
- Text-to-speech narration of content

## Content Categories and Justification

### 1. Educational Content
- **Constellation Information**: Astronomical facts and mythological stories
- **Astrological Data**: Information about zodiac signs and their meanings
- **Purpose**: Educational and entertainment value
- **Target Audience**: All ages interested in astronomy and astrology

### 2. Interactive Features
- **Location Services**
  - Used solely for astronomical calculations
  - Required for accurate constellation identification
  - No tracking or data collection beyond immediate use
  - Clear user permission requests

- **Text-to-Speech**
  - Used for accessibility and convenience
  - Narrates educational content only
  - No inappropriate language or content
  - User-controlled feature

- **User Input**
  - Zodiac sign selection
  - Constellation identification
  - No user-generated content
  - No social sharing features

### 3. Data Collection and Privacy
- **Stored Information**
  - Zodiac sign preference (local storage only)
  - No personal identifiers
  - No user accounts
  - No data sharing

- **Location Data**
  - Used in real-time only
  - Not stored or transmitted
  - Required for astronomical accuracy
  - Clear permission requests

### 4. Content Moderation
- **Content Sources**
  - Pre-written mythological stories
  - Astronomical data from reliable sources
  - AI-generated horoscopes with appropriate filters
  - No user-generated content

- **Content Review Process**
  - All content is pre-screened
  - AI-generated content is filtered for appropriateness
  - No dynamic content that could be inappropriate
  - Regular content updates reviewed for compliance

### 5. Safety Measures
- **Parental Controls**
  - No in-app purchases
  - No social features
  - No external links
  - No chat or messaging

- **Content Filtering**
  - AI-generated content is filtered for:
    - No profanity
    - No adult content
    - No violent content
    - No gambling references

### 6. Technical Implementation
- **Data Storage**
  - Local storage only
  - No cloud synchronization
  - No user accounts
  - No data transmission

- **Permissions**
  - Location (for astronomical calculations)
  - Text-to-speech (for narration)
  - No other permissions required

## Compliance with Rating Guidelines
